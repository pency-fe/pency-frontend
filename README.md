## 기획

- 웹툰: 시리즈 단위 (장르, 페어, 창작 유형, 단편/연재/완결, 연령 제한), 무조건 강제!!! -> 똑같해
- 웹소설: 시리즈 단위 (장르, 페어, 창작 유형, 단편/연재/완결, 연령 제한), 무조건 강제!!! -> 똑같해

- 자유 창작 (알아서해, 장르 없어. 알아서 해(웹툰, 웹소설 빠진 포스타입)) -> 알아서 해 -> 태그 추천 정도만 넣어주면
  - 포스트: 지금 웹툰 처럼 -> 알아서 해
  - 시리즈: 지금 웹툰 처럼 -> 알아서 해
